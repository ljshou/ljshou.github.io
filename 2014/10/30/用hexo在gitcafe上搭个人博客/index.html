Set up personal blog using hexo on github/gitcafe

Oct 30 2014
hexo

This blog tells you how to set up your own blog rapidly using Hexo on github/gitcafe pages.

Prerequisite

  1. install Node.js
    • https://nodejs.org/en/
  2. install git
    • Ubuntu: sudo apt-get install git
    • Windows: https://git-for-windows.github.io/
  3. install Hexo:
    • Ubuntu: sudo npm install -g hexo-cli
    • Windows: npm install -g hexo-cli

Hexo official site: Hexo

Set up blog locally

Hexo commands

  • Initialize hexo blog: hexo init
  • Write a new blog: hexo new "blog's title"
  • Generate static files: hexo generate
  • Blog preview: hexo server. preview blog at localhost:4000
  • Blog deploy: hexo depoly

Hexo deploy
